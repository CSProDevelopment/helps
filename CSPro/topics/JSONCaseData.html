cases

Full data for the file.

Array (>= 0 items)
key
required
string

The case key.

uuid
required
string

The case UUID (universally unique identifier).

label
string

The case label, if applicable.

deleted
boolean

Indicates whether the case is deleted.

verified
boolean

Indicates whether the case is verified.

object or null (partialSave)

Details on partial save, if applicable.

Array of objects (notes) >= 0 items

An array containing all notes for the case.

object (level-1)

Represents the top level of the data structure.
